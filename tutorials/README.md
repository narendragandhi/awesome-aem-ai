# AEM AI Hands-on Labs

This directory contains hands-on tutorials for working with AI tools in Adobe Experience Manager.

## Labs Overview

| Lab | Topic | Duration | Prerequisites |
|-----|-------|----------|---------------|
| **Lab 1** | Setting Up AEM MCP Server | 20-30 min | Node.js 18+, AEM instance |
| **Lab 2** | Experience Production Agent | 25-35 min | AEM Cloud + AI Agents enabled |
| **Lab 3** | Claude Code Skills for AEM | 30-40 min | Claude Code, AEM/EDS project |
| **Lab 4** | ACM MCP Server | 30-40 min | Maven, Java 17+, AEM with ACM installed |

## Getting Started

### Prerequisites Checklist

- [ ] Node.js 18+ installed
- [ ] AEM as a Cloud Service access (or local AEM SDK)
- [ ] Claude Code installed
- [ ] GitHub CLI installed (for EDS skills)
- [ ] Maven 3.9+ and Java 17+ (for Lab 4)

### Recommended Order

1. **Lab 1** - Foundation: Learn to set up MCP server for AEM connectivity
2. **Lab 2** - Automation: Use AI agents for content operations
3. **Lab 3** - Development: Enhance coding with Claude Code skills

## Quick Start

```bash
# Clone this repository
git clone https://github.com/narendragandhi/awesome-aem-ai.git
cd awesome-aem-ai/tutorials

# Follow the lab that matches your needs
cat lab-1-aem-mcp-server.md
```
